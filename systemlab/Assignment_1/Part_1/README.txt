Compilation and Running:
The command to compile the run the file use following command:

make

Explaination :

->The program reads the file arrival.txt (File can be changed in Source.cpp)

->The burst time for each process is calculated using the rand function.

->The function than processes the processes using the Preemptive Shortest Time first.

->The Result is than printed using Gantt chart which in turn is stored in file 'Output.txt'
