Compilation and Running:
The command to compile the run the file use following command:

Environment : Linux System

make

Explaination :

->Program to Implement Banker's Algorithm.

->The Program first Reads from the "input.txt" the following Matrices :

    a. Number of Processes and No of Resources.
    b. Current Available Resources
    c. Pre-Allocated Resources to the processes.
    d. Max Allocation required by each process.

->It later finds the "Need_Allocation" Matrix to find the extra required want of each process.

->Finds a Safe Sequence of the given Input.

->Create threads to Enact the sequence and Terminate.

->If no Safe Sequence than reports it and Terminates.
